# 🗑️ Waste Bin Classification GUI

A simple Python GUI application that classifies waste into appropriate bins (Wet, Dry, and E-Waste) based on user input.

## 🚀 Features
- Built using Tkinter
- Classifies waste into:
  - Wet Waste → Green Bin
  - Dry Waste → Blue Bin
  - E-Waste → Gray Bin
- Instant result display
- Color indicator for bin type
- Beginner-friendly project

## 🛠️ Requirements
- Python 3.x

   python simple_waste_bin_gui.py

## 🧠 How It Works
- Enter a waste item (e.g., "banana peel", "phone")
- The program checks keywords
- Displays the waste category and corresponding bin color

## 📌 Example
- banana peel → Wet (Green Bin)
- phone charger → E-Waste (Gray Bin)
- plastic bottle → Dry (Blue Bin)

## ⚠️ Note
This is a basic project using keyword matching (no AI/ML).
